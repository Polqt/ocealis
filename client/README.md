# Ocealis client

SolidStart UI for the anonymous shared living ocean.

```bash
pnpm install
pnpm dev
```

Optional env:

```bash
VITE_API_URL=http://localhost:8080
```

The first viewport is the ocean: brand, one line of promise, cast CTA, and the live Three.js sea. Bottle panels open for discover / journey chapters / re-release.
